# Melanoma Skin Cancer Detection Using Transfer Learning

This repository contains code, notebooks, and experiments for detecting melanoma using transfer learning.

## Overview
- Dataset: ISIC / HAM10000 (store raw images in `data/raw/`)
- Models: EfficientNet, ResNet, MobileNet
- Tasks: preprocessing, training, evaluation, Grad-CAM visualization

## Quickstart
```bash
git clone <repo-url>
pip install -r requirements.txt
python src/models/transfer_model.py
jupyter notebook
```
